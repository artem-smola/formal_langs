# Города и дороги — решение на C++

Программа распределяет города по государствам по алгоритму из условия:
1. Изначально каждое государство содержит свою столицу.
2. Затем по очереди (1..k, циклически) каждое государство добавляет ближайший
   незанятый город, который имеет дорогу хотя бы к одному городу этого государства.
3. Процесс продолжается, пока все города не будут распределены.

## Формат ввода

Из `stdin`:

1. `n m` — число городов и число дорог.
2. `m` строк: `i j len` — дорога между городами `i` и `j` длиной `len`.
3. `k` — число государств (столиц).
4. `k` чисел — номера столиц.

Ограничения по условию:
- граф дорог связный;
- нумерация городов: `1..n`.

Если граф несвязный или вход обрезан/поврежден, программа завершится с ошибкой.

## Формат вывода

Для каждого государства:

`Государство X: c1 c2 c3 ...`

Где `X` — номер государства (по порядку столиц во входе),
а далее список городов этого государства в порядке присоединения
(сначала столица).

При равенстве длин используется tie-break:
1) меньший номер города, 2) меньший номер города-источника.

## Сборка

```bash
make
```

## Запуск

```bash
./cities < input.txt
```

или

```bash
make run < input.txt
```

## Очистка

```bash
make clean
```
