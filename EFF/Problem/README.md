# EFF — постановка задачи

Кратко: для КС-грамматики требуется построить множество `EFF_k(alpha)`:

1. Рассматриваются терминальные цепочки, получаемые **правосторонними** выводами из `alpha`.
2. Для каждой терминальной цепочки берется:
   - вся цепочка, если ее длина меньше `k`;
   - иначе префикс длины `k`.
3. В `EFF_k(alpha)` включаются только те листья вывода, у которых
   последнее примененное правило не является `epsilon`-продукцией.
