# Спутники и группировки — решение на C++

Программа строит группировки КА по тактам по 10 минут на глобальном интервале времени.
Для каждого такта формируется граф активных связей, после чего группировки считаются как компоненты связности.

## Формат ввода

Поддерживаются два формата:

1. `n m`
2. `m` строк: `i j start finish`

или

1. `n m T`
2. `m` строк: `i j len start finish`

Где:
- `i`, `j` — номера КА (`1..n`);
- интервал активности всегда трактуется как полуинтервал `[start, finish)`;
- в формате с `len` требуется строго `len = finish - start`;
- `T` (если задан) ограничивает интервалы сверху: `finish <= T`.
- если `T` не задано, оно автоматически берется как максимум `finish` по всем связям.

## Формат вывода

Для каждого такта `[t, t+10)`:

`Такт [t, t+10):`

`  Группировка X: ...`

В группировках перечисляются номера КА.

## Машинный формат вывода

Для автопроверки можно использовать:

```bash
./ka_groups --machine < input.txt
```

Формат:

- `tick t0 t1 groups g`
- `group id size v1 v2 ...`

## Сборка


## Запуск

```bash
./ka_groups < input.txt
```

или

```bash
make run < input.txt
```
